# Example Widget

This folder is a copyable starter for a new module widget without auto-registering it into Studio.

## How To Use It

1. Copy these files into a real widget folder.
2. Remove the `.txt` suffix from each template.
3. Rename `promo-pill` to your widget name.
4. Register the final definition in the real module definitions index only after the files compile and render.

The templates stay as plain text on purpose so they do not change runtime registration, audit metrics, or bundle output by accident.
