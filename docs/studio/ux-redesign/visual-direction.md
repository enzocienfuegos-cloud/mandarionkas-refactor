# Studio Visual Direction

## Intent

Studio should feel premium, sharp, and creative-forward.

The product should signal:

- confidence
- speed
- editorial clarity
- polished interaction
- strong hierarchy

## Palette roles

- `fuchsia`: CTA, selection, active controls
- `violet`: premium depth, gradients, secondary emphasis
- `cyan`: guides, helper overlays, snapping/data affordances
- `green`: success
- `amber`: warning
- `red`: blocking error

## Surface language

- Glass and panel surfaces should feel layered, not muddy.
- Cards should separate title, meta, preview, and next action clearly.
- Empty states should guide action, not just explain absence.
- Primary CTA should be obvious within 2 seconds on hub surfaces.

## Interaction language

- Hover should communicate hierarchy and clickability.
- Focus must remain strong and accessible.
- Selected states should use premium accent rings rather than operational green.
- Motion should stay subtle and intentional.
